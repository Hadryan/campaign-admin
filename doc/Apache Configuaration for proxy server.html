• make sure the Apache curl module enable.

    If curl module is not enabled in the php, please install with following command.

    sudo apt-get install php5-curl

    you can confirm the curl module with this code in the *.php file

    
                <?php
                   phpinfo();
                ?>

    I've added info.php file in the project path.

    http://[servername]/info.php

  • change directory access permission.

    In the /opt/bitnami/apache2/conf/httpd.conf file uncomment the following line.

    Include conf/extra/httpd-vhosts.conf

    In the /opt/bitnami/apache2/conf/extra/httpd-vhosts.conf add the following codes.

    
                <VirtualHost *:80>
                    ServerName 54.252.212.90
                    DocumentRoot "/opt/bitnami/apps/backbone-campaign-admin"           // set the document root to the project path
                    <Directory "/opt/bitnami/apps/backbone-campaign-admin">            // give access permission to the project path
                        AllowOverride All
                        Order allow,deny
                        Allow from all
                    </Directory>
                </VirtualHost>

  • restart server.

    sudo /[installdir]/ctlscript.sh restart apache
